2System Prompt Best Practices
Python — Production Agent System Prompt

SYSTEM_PROMPT = """You are a specialized NVIDIA technology assistant.

## Role
Help engineers deploy and optimize AI models using NVIDIA tools.

## Constraints
- Only answer questions about NVIDIA technology stack
- If you are unsure, say "I don't know" rather than guessing
- Always cite the specific NVIDIA tool or documentation when relevant
- Maximum response length: 300 words

## Output Format
When explaining a technical process:
1. Start with a one-sentence definition
2. Explain why it matters
3. Give a concrete code example
4. Note any gotchas or common mistakes

## Tone
Technical, precise, direct. No unnecessary hedging."""

from openai import OpenAI
client = OpenAI(base_url="https://integrate.api.nvidia.com/v1", api_key="nvapi-xxx")

response = client.chat.completions.create(
    model="meta/llama-3.1-70b-instruct",
    messages=[
        {"role": "system", "content": SYSTEM_PROMPT},
        {"role": "user", "content": "What is the gRPC port for Triton?"}
    ]
)
3Structured Output for Agents
Python — JSON Structured Output

from pydantic import BaseModel
from openai import OpenAI

class TritonConfig(BaseModel):
    model_name: str
    grpc_port: int
    http_port: int
    metrics_port: int
    backend: str

client = OpenAI(base_url="https://integrate.api.nvidia.com/v1", api_key="nvapi-xxx")

response = client.beta.chat.completions.parse(
    model="meta/llama-3.1-70b-instruct",
    messages=[
        {"role": "user", "content": "Give me default Triton server config as JSON."}
    ],
    response_format=TritonConfig
)
config = response.choices[0].message.parsed
print(config.grpc_port)  # 8001

🎯 Exam Key Points — Cognition & Planning (10%)
  • Zero-shot: task description only. Few-shot: examples included. CoT: explicit reasoning steps
  • System prompt defines agent persona, constraints, and response format
  • Structured output: use response_format or JSON schema in prompt for parseable output
  • Few-shot prompting improves consistency for novel formats vs zero-shot
  • Temperature: 0.0 for deterministic tool calls, 0.7 for creative responses
