Chain-of-Thought & Planning

Advanced reasoning patterns for agents — Chain-of-Thought (CoT), Self-Consistency, Tree-of-Thought (ToT), Monte Carlo Tree Search (MCTS), R-MCTS, and 2025 planning frameworks (ReAct-Plan, Plan-and-Act).

1Chain-of-Thought (CoT)

CoT prompting improves LLM reasoning by asking the model to show intermediate reasoning steps before giving a final answer. Simply adding "Let's think step by step" or providing step-by-step examples significantly improves accuracy on math, logic, and multi-hop reasoning tasks.

VariantMechanismWhen to Use
Zero-shot CoT"Think step by step" in promptQuick improvement, no examples needed
Few-shot CoTProvide 3–8 step-by-step examplesConsistent format, complex tasks
Self-ConsistencySample CoT N times → majority vote answerHigh-stakes decisions, reduces variance
2Tree-of-Thought (ToT)

ToT treats reasoning as a search problem over a tree of "thoughts". At each step, the LLM generates multiple possible next reasoning steps (branches), evaluates which are most promising (pruning), and explores the best path. This enables deliberate exploration — looking ahead and backtracking — that linear CoT cannot do.

Best for: strategic planning, creative problem-solving, complex multi-step tasks where linear reasoning fails.

3Monte Carlo Tree Search (MCTS) & R-MCTS

MCTS is a heuristic search algorithm that balances exploration (trying new paths) and exploitation (deepening known-good paths) using random sampling (Monte Carlo rollouts). Integrated with LLMs, it enables agentic planning in large, uncertain decision spaces.

R-MCTS (Reflective MCTS) (2025): extends classic MCTS by adding reflection — each node stores lessons from previous rollouts. Uses multi-agent debate for reliable state evaluation. Demonstrated strong results on WebArena (web navigation) benchmarks.

MCTS 4 phases: Selection → Expansion → Simulation → Backpropagation
