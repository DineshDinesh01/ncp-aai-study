NeMo Toolkit

NVIDIA's end-to-end framework for building, training, and fine-tuning large language models — NeMo Launcher, Megatron-LM backend, and the five NeMo microservices (Customizer, Evaluator, Guardrails, Retriever, Curator).

1What Is the NeMo Toolkit?

NVIDIA NeMo Toolkit is an open-source framework for building LLMs from scratch or fine-tuning existing ones at scale. It uses Megatron-LM as its training backend, which provides tensor parallelism (split model layers across GPUs) and pipeline parallelism (split model stages across nodes) for multi-GPU, multi-node training.

⚠️ NeMo Toolkit ≠ NeMo Guardrails

These are two separate tools. NeMo Toolkit = training and fine-tuning framework. NeMo Guardrails = safety policy enforcement toolkit. Confusing them is a common exam mistake.

2NeMo Microservices (GA April 2025)

The NeMo ecosystem now includes five managed microservices forming a "data flywheel" — continuously improving agents through evaluation-driven iteration:

MicroserviceRole in PipelineKey Capability

4NeMo Customizer API Call
Python — NeMo Customizer REST API

import requests

# Create a fine-tuning job
job = requests.post(
    "https://customizer.api.nvidia.com/v1/customization/jobs",
    headers={"Authorization": "Bearer nvapi-xxx"},
    json={
        "model": "meta/llama-3.1-8b-instruct",
        "method": "lora",
        "hyperparameters": {
            "training_type": "sft",
            "finetuning_type": "lora",
            "lora": {"adapter_dim": 16, "alpha": 32},
            "epochs": 3,
            "learning_rate": 0.0002
        },
        "dataset": {"name": "my-agent-dataset", "format": "sharegpt"}
    }
).json()

print(job["id"])  # job-id for polling status
