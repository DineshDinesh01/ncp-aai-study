🔬 Hands-On Labs

8 Jupyter labs that build real skills for the NCP-AAI exam and production work. Each lab uses the NVIDIA NIM API (free at build.nvidia.com) or runs algorithmically without any API key.

🚀 Getting Started

1. Get a free NVIDIA NIM API key at build.nvidia.com
2. Click "Open in Colab" on any lab — runs in your browser, no local GPU needed
3. Set NVIDIA_API_KEY as a Colab secret (🔑 icon → New secret)
4. Run all cells — each lab is fully self-contained

Lab 001 — Transformer Attention Visualization
🔮
Transformer Attention Visualization
Agent Development Intermediate
Extract and visualize attention matrices from GPT-2. Compare causal masking (GPT-style) vs bidirectional attention (BERT-style). Call NVIDIA NIM for comparison. Understand why multi-head attention works.
⏱ ~45 min 📦 transformers, matplotlib 🔑 Optional: NIM API
📓 Open in Colab 📖 Topic Notes
Lab 002 — BPE Tokenizer from Scratch
🔤
BPE Tokenizer from Scratch
Agent Development Beginner
Implement the BPE merge algorithm in ~60 lines of Python (no libraries). Tokenize English, code, and non-English text. Compare with HuggingFace GPT-2 tokenizer. Measure how vocabulary size affects token efficiency.
⏱ ~30 min 📦 transformers (comparison only) 🔑 No API key needed
📓 Open in Colab 📖 Topic Notes ⚡ Try in Playground
Lab 003 — Prompt Engineering Comparison
✍️
Prompt Engineering Systematic Comparison
Cognition & Planning Beginner
Side-by-side comparison of zero-shot vs few-shot vs Chain-of-Thought on 5 evaluation tasks using the NVIDIA NIM API (meta/llama-3.1-70b-instruct). Measure accuracy, consistency, and token cost for each approach.
⏱ ~40 min 📦 openai, pandas 🔑 NVIDIA NIM API required
📓 Open in Colab 📖 Topic Notes
Lab 004 — ReAct Agent from Scratch
🤖
Build a ReAct Agent from Scratch (80 lines)
Agent Architecture Intermediate
Implement the full Thought → Action → Observation loop without using LangChain or AutoGen. Wire up 3 tools (Calculator, Wikipedia search, Weather API mock). Trace and visualize the reasoning steps. See exactly how tool calling works internally.
⏱ ~60 min 📦 openai only 🔑 NVIDIA NIM API required
📓 Open in Colab 📖 Topic Notes
Lab 005 — LangGraph HITL Workflow
🔀
LangGraph Stateful Agent with Human Approval Gate
Agent Architecture Advanced
Build a 3-node LangGraph (draft → human_review → send). Use interrupt() to pause, present draft to human, and resume with Command(resume=...). Implement MemorySaver checkpointing. Demonstrate time-travel debugging by replaying from a past checkpoint.
⏱ ~75 min 📦 langgraph, langchain-nvidia-ai-endpoints 🔑 NVIDIA NIM API required
📓 Open in Colab 📖 LangGraph Notes 📖 HITL Notes
Lab 006 — RAG Pipeline End-to-End + RAGAS Evaluation
🗄️
Full RAG Pipeline with RAGAS Evaluation
Knowledge Integration Advanced
Build a complete RAG system: PDF → PyPDF chunk → SentenceTransformer embed → ChromaDB store → retrieve → NIM generate. Evaluate with RAGAS (Context Precision, Recall, Faithfulness, Answer Relevancy) on 10 Q&A pairs. Tune chunk size and k and see metric impact.
⏱ ~90 min 📦 chromadb, ragas, sentence-transformers, pypdf 🔑 NVIDIA NIM API required
📓 Open in Colab 📖 RAG Notes 📖 RAGAS Notes
Lab 007 — Multi-Agent System with AutoGen
👥
3-Agent AutoGen System: Researcher + Coder + Critic
Agent Architecture Advanced
Build a 3-agent AutoGen GroupChat: UserProxy (orchestrator) + AssistantAgent (researcher) + CriticAgent (review). Agent writes and executes Python code, Critic validates output, loop runs until task is correct. Implement a semantic router in front to dispatch queries to the right agent.
⏱ ~75 min 📦 autogen, sentence-transformers 🔑 NVIDIA NIM API required
📓 Open in Colab 📖 Multi-Agent Notes
Lab 008 — NeMo Guardrails: Colang Safety Rails
🛡️
NeMo Guardrails: Build Topical + Input + Output Rails
Safety & Compliance Intermediate
Install NeMo Guardrails, write Colang rail files for: topical rail (block off-topic queries), input rail (detect and reject harmful prompts), output rail (prevent PII leakage). Test with 10 adversarial inputs. Measure block rate and false-positive rate.
⏱ ~60 min 📦 nemoguardrails, openai 🔑 NVIDIA NIM API required
📓 Open in Colab 📖 Guardrails Notes
